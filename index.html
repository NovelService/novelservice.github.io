What is this?

Novel Service is a web service with which you can download any website as an ebook in epub format

You can combine any number of websites into a single ebook.

How to use?

Simply write your url into the big textfield for urls. If you want to combine multiple sites write one url per line. Then submit and wait until the download starts.

After your request is submitted, it takes at least 2 seconds per url. Either wait here or save the id and come back later.
